Creating spanned rows and columns using tables - Once you have selected the th scope="col" and th scope="row" you can then use the table data tag to merge table data using the td colspan="2" or td rowspan="2" tags

Table borders - By default no borders show in tables and when you select borders using ccs styling the table will have a gap between in cell. To bring the table borders together you can the enter the following css styling: table, th, td{
border-collapse: collapse;
border: solid black 1px;
}

You can select individual table borders using css styling example border-top: 1px solid black;.

Adding background colour - You can also add background colour using css styling to cells

Create padding in table cells - By default there is no table spacing between text and table cells. You can change this by adding padding using css styling

Create cell spacing - If you want to create cell spacing you can use the following code: border-spacing 1em;. Note you use the element table and th td in css

Table widths - By default table cells fill by the size of content. You can change the table cells by selecting the following css: th td {
width =20%;
}
You can also change the table size using the following css styling: table {
width: 75%;
} Make sure that the width size is less than 100%

Table text align - You can change the text alignment using css. For instance you can use text-align: left, right, center, and justify. You can also change the vertical-align: top, center, or bottom.
